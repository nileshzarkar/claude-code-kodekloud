What is a METAR? A METAR (Meteorological Aerodrome Report) is the standard format for current weather observations at airports, issued every hour (or more often when conditions change rapidly). Pilots use METAR reports for pre-flight planning, approach briefings, and in-flight weather awareness. Each METAR follows a strict, fixed order: Station → Date/Time → Wind → Visibility → Weather → Clouds → Temperature → Pressure → Remarks.
